# Ghost Repo

[![Website](https://img.shields.io/badge/Website-ghostrepo.vercel.app-blue)](https://ghostrepo.vercel.app)

**Ghost Repo** is a lightweight platform for sharing GitHub repositories privately and unlisted. Perfect for developers, collaborators, or teams who want to share code without making it public.

- **Realtime Analytics:** Track repository access and activity in real-time.
- **Easy Integration:** Simple, intuitive interface for sharing any GitHub repo link.
